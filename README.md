# es6-tutorial

es6语法整理，这里整理的是最基础最常用的es6知识点，帮你彻底弄懂es6。  

对于不常用的字符串扩展(模板字符串除外)，数值扩展这里没有作代码演示，[点击这里可以查看系列文章](https://www.jianshu.com/p/492c660c9374)

## 使用

使用gulp把es6编译成es5的代码，方便查看es6与es5的差异

```
npm install -g gulp  

npm install  

gulp/npm run build 

```
